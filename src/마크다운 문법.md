# 마크다운 사용법(문법)

## 헤더
- "#"의 개수를 늘러가면 제목1, 제목2 로 늘려갈 수 있음
- "=" 혹은 '-' 사용

## 강조
- 이텔릭체: *별표 양쪽에 하나씩* 혹은 _언더바 양쪽에 하나씩_
- 볼드체: **별표 양쪽에 두개씩** 혹은 __언더바 양쪽에 두개씩__
- 취소선: ~~양쪽에 물결 두개씩~~
- 밑줄: <u>밑줄은 "앞쪽: <u>, 뒷쪽: </u> </u> 사용
  - 밑줄은 마크다운에서 지원하지 않기 때문에 '<u></u>' 태그 사용
